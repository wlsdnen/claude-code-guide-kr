# docs/ 에이전트 가이드

15개 섹션별 마크다운 문서 디렉토리.

## 파일 규칙
- 파일명: `{번호}-{주제}.md` (01~15)
- 상단 네비게이션: `[← 목차로 돌아가기](../README.md) · [이전: {제목}]({파일}) · [다음: {제목} →]({파일})`
- 하단 네비게이션: `[다음: {제목} →]({파일})`
- 제목은 `# ` (h1) 하나만 사용

## 내용 규칙
- 한국어로 작성
- 코드 블록에 언어 태그 명시 (```json, ```bash, ```yaml 등)
- 테이블은 GitHub Flavored Markdown 형식
- examples/ 파일 참조 시 상대 경로 사용: `[예시](../examples/파일명)`
